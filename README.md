# my-project
Full-feature ice cream ordering website with admin panel, login system, and review management.  Cute and interactive ice cream ordering system built with PHP &amp; MySQL.
User Side

Browse ice cream flavors with a cute & modern UI
Add to cart & place orders
Login/Signup system
Track order status (Pending → Completed)
Submit and view customer reviews

Admin Panel
Admin login authentication
Manage ice cream flavors (Add / Edit / Delete)
Manage customer orders (Update status)
Manage user accounts
Manage customer reviews
Dashboard with stats (Users, Orders, Top Flavors)

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Design: Cute pastel theme + mobile-first layout

How to Run
Download the project
Place it in your local server directory (XAMPP → htdocs)
Import database.sql into phpMyAdmin
Update database credentials in config.php (if needed)
Run the website via localhost/icecream-website
